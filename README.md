# Repositório para Análise de dados, Treinamento de Modelo preditivo e Análise de Resultados para dados obtidos da NASA sobre Eclipses Lunares e Solares
## Bibliotecas utilizadas
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Sklearn
- Pycaret
- Figlet
